# P.A.C.E. – Power Automotive Centre of Excellence

Official website for **Power Automotive Centre of Excellence (P.A.C.E.)**, a family-owned automotive repair shop based in **Calgary, Alberta**.

This site is designed to be:
- Clean and professional
- Easy to read for an older audience
- Simple to update and deploy via **Netlify**
- Scalable for future features (online booking, staff portal, Square integration)
